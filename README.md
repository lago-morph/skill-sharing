# skill-sharing

A design effort for `skillctl` — a Python CLI that lets a small team inventory, share, merge, and selectively customize AI agent skills across multiple coding-assistant platforms (Claude Code, Codex, Cursor, Aider, …).

This repo is currently in the **design phase**. No code yet.

- [`docs/decisions.md`](docs/decisions.md) — locked-in design decisions with rationale.
- [`docs/prior-art.md`](docs/prior-art.md) — landscape: what exists, what we'll build on, where this earns its keep.
- [`docs/iteration-plan.md`](docs/iteration-plan.md) — six iterations sized for a 4-person team, each independently shippable.
- [`docs/skill-schema.md`](docs/skill-schema.md) — proposed skill format (frontmatter + section schema).
- [`docs/research-brief.md`](docs/research-brief.md) — self-contained brief to feed to a deep-research agent for an external review of these decisions.
